# Transform — Audio Transcription

Десктопний застосунок для транскрипції аудіо в текст за допомогою AI (OpenAI Whisper).

## Можливості

- Підтримка форматів: mp3, mp4, m4a, wav, ogg, flac, webm
- 12 мов (українська, англійська, німецька та інші)
- Моделі Whisper: tiny, base, small, medium, large
- Обрізка аудіо по секундах
- Результат зберігається у .txt файл поряд з аудіо

## Запуск

```bash
pip install -r requirements.txt
python transcribe.py
```

## Вимоги

- Python 3.8+
- ffmpeg (додати в PATH)
- Залежності з requirements.txt


Ось вимоги до проекту:

1.  Методологія — обов'язкова, без винятків:
Дослівно. Жодних синонімів, жодних скорочень, жодного "згладжування". Переклад повинен відображати точний характер мовлення, включно з незакінченими реченнями, самоперебиваннями, повторами, суржиком, русизмами, розмовними кліше ("тіпа", "корочє", "ось" тощо) — передавай відповідними розмовними еквівалентами англійської.
Мат і лайка — перекладаються прямо, у відповідному регістрі, без евфемізмів і зірочок.

2. Якщо слово або вираз не перекладається адекватно — залишай оригінальне українське слово.

3. Нічого не додавати, нічого не прибирати. Жодних логічних «продовжень», уточнень або домислювань від себе. Якщо текст обривається або звучить незавершено — перекладати рівно те, що є.
